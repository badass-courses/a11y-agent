# a11y-agent

Accessibility skills for AI coding agents — 1 core skill, 2 commands (/a11y-audit, /a11y-fix).

Covers WCAG 2.2 AA: semantic HTML, keyboard accessibility, ARIA, color contrast, focus management, forms, motion safety, and automated testing.

## Commands

- `/a11y-audit [area]` — Audit current component or page for accessibility issues. Produces a prioritized report with WCAG references.
- `/a11y-fix [area]` — Fix accessibility issues in current code. Explains rationale and flags what to test manually.

## Core Skill

The `a11y-agent` skill loads automatically and provides deep accessibility knowledge for every code review and implementation task.
